About Me

My name is Cisco Rendon, I have been web developing since 2013. Started like everyone else, curious on how the internet and web pages worked. My very first web page was created back in the days of the good ol' AOL craze. I went to the Science and Business Library here in New York City, borrowed a book and created my first website. It was never published because I did it only as curiosity of how things worked. Who would've thought that many years later, I would make a career out of curiosity.

I have since grown in the days of technology, where everyone is encouraged to learn to code. I know start all my custom projects with a wireframe of what the website is supposed to look like on a PSD or similar and work it into a mockup using HTML5, CSS3, PHP & Javascript. Once approved, it is converted into a powerful WordPress Custom Theme. This portfolio page is only a small glimpse of what I can do using all the tools at my disposal, using a custom boiler plate template I created for myself using NodeJs. Yes, it works with PHP and maybe one day, I will put it up for others to use as well. Only time will tell.
